#  FitMind AI - Personal Health Coach

FitMind AI is an AI-powered web application that analyzes user health data and provides intelligent insights, predictions, and personalized health advice.

---

##Features

* Health Score Calculation
* Future Health Prediction using Machine Learning
* AI-powered Health Advice (LLM Integration - Groq API)
* Upload CSV Health Data
* Data Analysis using Pandas
* Interactive Web Interface (Flask)

## Tech Stack

* **Backend:** Python, Flask
* **Data Processing:** Pandas, NumPy
* **AI Integration:** Groq API (LLM)
* **Frontend:** HTML, CSS
* **Visualization:** Matplotlib / Seaborn
## How It Works

1. Upload your health dataset (CSV)
2. System analyzes:
   * Heart rate
   * Oxygen levels
   * Activity & sleep
3. Generates:
   * Health score
   * Predictions
   * AI-based suggestions
